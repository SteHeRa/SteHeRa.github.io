DEEPSEE

DEEPSEE is an app designed to help scuba divers make informed decisions about their next dives and aims to replace the conventional divers log book.
This is achieved by providing a database of user submitted images & metadata of dives that have been recently logged all packaged in a sleek and functional UI.

  • Built the app from the ground up with a user-friendly, functional design.
  • Tech stack: React, Mapbox, Mongoose, Express, Cloudinary.
